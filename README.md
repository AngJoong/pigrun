# Pig Run Pinball

핀볼 테이블처럼 생긴 장애물 코스에서 여러 돼지가 자동으로 달리는 관전형 레이스 게임입니다. 사용자는 직접 조작하지 않고, 시작 전에 1등할 돼지를 고릅니다.

## 플레이

1. 우승할 것 같은 돼지를 선택합니다.
2. `출발`을 누릅니다.
3. 첫 장애물인 나무 회전문을 지나 자동으로 달리는 결과를 봅니다.

## 현재 구현

- 6마리 돼지 자동 레이스
- 우승 예측 선택
- 첫 장애물: 나무 축과 네 개 팔이 도는 회전문
- 회전문은 튕겨내는 범퍼가 아니라 돼지를 막고, 팔이 지나가 길이 열리면 통과시킴
- 중간 마름모 구역은 실제 울타리 충돌로 막혀 있고, 돼지는 위/아래 갈래길로 돌아가야 함
- 마름모 오른쪽 꼭지점 뒤는 점점 좁아져 한 줄 길이 되고, 그 안에 작은 회전문들이 지그재그로 배치됨
- 뒤쪽 코스에는 S자 돼지우리 울타리, 진흙/먹이 선택 구간, 결승 전 3문 게이트가 이어짐
- 돼지는 앞쪽 회전문과 마름모 울타리를 보고 위/아래 경로를 선택함
- 회전문에 닿은 돼지는 코/이마로 문을 밀고 있는 자세가 됨
- 돼지는 고정 레인 없이 초원 안에서 자유롭게 위아래로 이동
- 트랙 위/아래 경계에 닿으면 튕기며 떼굴떼굴 구르다가 다시 달리는 자세로 복귀
- 실시간 순위와 카메라 추적
- 별도 빌드 없이 `index.html` 실행
